SUPPORTED ACTIONS
Close Window :: Close a target window
Control >> End :: Skip all remaining actions of the current schedule
Control >> Disable Schedule Manager :: Stop all running schedules
Delay :: Pause before executing next action
IfWindowOpen :: Find window. Fail if window is not found.
IfWindowNotOpen :: Find window. Fail if window is found.
MouseClickAbsolute :: Switch to window and mouse click position by absolute co-ordinates.
MouseClickRelative :: Switch to window and mouse click position by relative to top-left corner of window.
NewProcess :: Run command
OpenURL :: Open a URL
Power >> Shutdown :: Shutdown the computer
Power >> Restart :: Restart the computer
Power >> Logoff :: Logoff the computer
StopSchedule :: Stop a schedule within the same group
StartSchedule :: Start a schedule within the same group
SwitchWindow :: Switch a target window
